# WePay – AI Powered Digital Wallet

WePay is a fintech-style digital wallet application built using React and Vite.  
It demonstrates secure digital transactions, wallet management, and AI-powered features.

## Features
- Digital wallet balance
- Transaction history
- AI security layer for fraud detection
- Offline and online transaction mode
- Responsive UI built with Tailwind CSS

## Tech Stack
- React
- Vite
- Tailwind CSS
- Axios
- Zod validation
- AI SDK
